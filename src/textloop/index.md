## API

|   名称    |     描述    |     类型     |      默认值       |

|   type    |  文字滚动方向 | boolean   |   true(横向滚动)    |
|   speed   |   滚动速度    |  number   |                 | 
|   style   |   额外的样式名  |   React.CSSProperties |   --   |
|   warpperClassName  | 外层样式名称  |  string   |  --   |
